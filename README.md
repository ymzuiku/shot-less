# Clean Screenshots（清理截图）

Quickly find, review, and bulk delete every screenshot on your iPhone or iPad.

一款帮助你快速找到、预览并批量删除 iPhone / iPad 相册中所有截图的工具。

## Features

- **Grouped by Date** — Screenshots organized chronologically so you always know where they came from
- **Auto Select** — Rapidly highlight screenshots in sequence; pause anytime before deleting
- **Batch Delete** — Select multiple screenshots and delete them all with one tap
- **Full-Screen Preview** — Tap to view full screen, swipe to browse, pinch to zoom up to 5×
- **Pinch to Resize Grid** — Adjust grid columns (1–6) to browse at your own pace
- **100% Private** — Everything happens on your device; no accounts, no uploads, no tracking

## Requirements

- iOS 15.0+
- Xcode 15+

## Build

```bash
make build
```
